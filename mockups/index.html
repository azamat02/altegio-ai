Автоматизация

Как использовать

  1. Открыть каждый мокап в Chrome / Safari, сделать скриншоты (Cmd+Shift+4 для области или полноэкранный).
  2. Мокапы 01-02 — iPhone frame уже встроен, скриншот делать область с телефоном.
  3. Мокап 03 (losses) — скриншотить целиком, это wow-материал для поста/звонка.
  4. Мокап 04 (PDF) — открыть в Chrome, Cmd+P → Сохранить как PDF → получится 3-страничный документ.
  5. Мокап 05 (лендинг) — залить на Vercel: cd mockups && vercel. Ссылку в рассылку.
